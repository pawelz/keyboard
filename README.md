# About

Configs for my keyboards. Unlikely to be useful for others, but also no need to
keep them private.

Those configs can be loaded using https://launcher.keychron.com/

## Keychron K6 Pro

Custom mappings:

```
Fn2+Home -> End
Fn1+Esc -> `
Fn2+Esc -> ~
Fn2+Backspace -> Del
Fn1+bulb -> Mouse3
Fn2+bulb -> Insert
(mac only) ROpt -> AltGr
```
